`archive_main_picture_face_blurred.png`

## Privacy note

Visible facial features were softened for public release. The blur is intended to reduce identifiability while preserving the grappling position, sequence context, timestamps, and evidence value. It does not guarantee anonymity because clothing, body shape, gym context, and source-video context can still be recognizable.
