## Option A: Portfolio / Data Analysis

I built a privacy-aware data analysis project around my YouTube learning archive.

The goal was not just to count videos. It was to turn messy personal export data into a reproducible analysis pipeline: API hydration, cleaning, feature engineering, rule-based classification, aggregate summaries, charts, and automated reports.

The final public version excludes raw data and row-level personal history. It only includes scripts, documentation, aggregate CSVs, charts, and reports.

What it demonstrates:

- API-based metadata enrichment
- Python data cleaning and feature engineering
- rule-based theme classification
- reproducible reporting
- privacy-aware public release design

The larger idea: personal data can become a learning-history map when handled carefully. The trick is making it useful without accidentally publishing your digital diary.
