## Known Data Quality Issues

| Issue | Detail |
|---|---|
| Tags missing | ~30% of videos have no tags. Classification relies on title and channel when tags are absent. |
| Like Count missing | ~13% of videos have no like count. YouTube allows creators to hide likes. |
| Deleted/private videos | 384 unique IDs (13.5%) returned unavailable. Content unrecoverable. |
| Duplicate rows | The master table contains one row per playlist entry, not one per video. The same video may appear in multiple playlists. |
| Duration of zero | A small number of entries (live streams, premieres) may have zero or missing duration. |
| Year column | Pre-computed in the takeout pipeline. Cross-checked against Added Timestamp during cleaning. |
