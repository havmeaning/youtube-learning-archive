# Project Manifest

This public package contains the GitHub-safe release of the YouTube Learning Archive Analysis project.

## Included

- Root project documentation
- Reproducible Python scripts
- Aggregate analysis CSVs
- PNG visualizations
- Markdown and HTML reports
- Publication safety checklist
- LinkedIn post drafts

## Excluded

- `Raw/`
- `Clean/`
- `master_table.csv`
- `master_table_hydrated.csv`
- `hydrated_videos.csv`
- `youtube_history_clean.csv`
- `youtube_history_themed.csv`
- API keys and environment files
- Full row-level personal history

## File Count

This package contains 42 files.

## Publication Note

Before publishing, run the safety checklist and inspect all CSV files. The included CSV files are aggregate summaries, not row-level personal history.
